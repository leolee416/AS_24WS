## Build project 
### Prerequsite
- ROS1
- catkin build tool
### Command
- run ```catkin build``` in ```/as_ws```.
- open terminator in directory of workspace.
- ```source devel/setup.bash``` or ```source devel/setup.zsh``` depends on you shell.
- run```roslaunch two_drones_pkg two_drones.launch```.
